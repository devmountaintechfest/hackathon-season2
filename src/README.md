# Setup
1. python -m venv venv
2. pip install -r requirements.txt

# How to run
- arg1 Execute type
  - M Migrate
- arg2 Datasource file
- arg3 Target db
- arg4 csv path
- arg5 Report name

Example
python run.py M ../data-devclub-1.xml ../database/devclub2022.db ../csv ../reports/data-devclub-report.json
